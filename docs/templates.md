# Templates

Templates let you start a note or board from a predefined structure instead of a blank page. They keep recurring documents — meeting notes, project briefs, weekly reviews — consistent across a team.

## Using a template

1. Create a new note and click **Use template** in the empty editor, or open the slash menu and choose **Template**.
2. Pick a template from the gallery or from your space's templates.
3. The note fills with the template's content, which you can then edit freely.

## Built-in templates

Tessera ships with a gallery of built-in templates, including **Meeting notes**, **Project brief**, **Weekly review**, **Product spec**, **Retrospective**, **Daily journal**, and **Reading list**. Built-in templates are available on every plan.

## Custom templates

Turn any note into a reusable template:

1. Open the note's **⋯** menu and choose **Save as template**.
2. Give the template a name and an optional description.
3. Choose whether it belongs to **you** or to the **space** (shared with all members).

Custom templates are available on the **Plus**, **Team**, and **Business** plans. The Free plan can use built-in templates but cannot create custom ones.

## Dynamic variables

Templates can include variables that fill in automatically when the template is used:

- `{{date}}` — today's date
- `{{time}}` — the current time
- `{{user}}` — the name of the person creating the note
- `{{space}}` — the current space name

For example, a meeting-notes template titled `{{date}} — Team sync` produces a note named "2026-06-30 — Team sync" when used.

## Template buttons

Insert a **template button** inside a note with the `/button` command. Clicking the button appends a block of predefined content — handy for adding a repeating checklist item or a standard section to a long-running document.

## Managing templates

Manage every template for a space under **Settings → Templates**. From there you can rename, duplicate, reorder, or delete templates, and set one as the **default** that new notes in the space start from.

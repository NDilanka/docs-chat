# Notes and the Editor

A note is the basic document in Tessera. The editor is a block-based, Markdown-friendly writing surface designed to stay out of your way.

## Blocks

Everything in a note is a **block**: a paragraph, heading, list item, image, table, or embed. Hover the left edge of any block to reveal the **handle** (⋮⋮). Drag the handle to reorder a block, or click it to open the block menu for actions like **Turn into**, **Duplicate**, and **Delete**.

Type **/** at the start of an empty line to open the **slash menu** and insert any block type.

## Markdown shortcuts

The editor understands Markdown as you type:

- `# `, `## `, `### ` create headings (levels 1–3).
- `- ` or `* ` start a bulleted list; `1. ` starts a numbered list.
- `[] ` creates a checkbox (a task).
- `> ` creates a quote; ``` creates a code block.
- `**bold**`, `*italic*`, `~~strikethrough~~`, and `` `code` `` format inline text.

## Headings and the outline

Headings build a navigable **outline**. Open it with the outline icon in the top-right of the editor or press **Cmd/Ctrl + Shift + O** to jump between sections of a long note.

## Tables, images, and files

Insert a table from the slash menu. Tables support up to **50 columns**. Drag an image or file directly into a note to attach it. Maximum upload size depends on your plan, from **50 MB** on Free up to **500 MB** on Business. Supported image types are PNG, JPG, GIF, WebP, and SVG.

## Code blocks

Code blocks offer syntax highlighting for over 100 languages. Set the language from the dropdown in the block's top-right corner. Use **Cmd/Ctrl + Shift + C** to copy a block's contents.

## Word count and reading time

Open the note's **⋯** menu and choose **Note info** to see word count, character count, and estimated reading time. A single note can hold up to **5 MB** of text content.

## Focus mode

Press **Cmd/Ctrl + .** to hide the sidebar and context panel for distraction-free writing. Press it again to bring them back.

## Trash and restore

Deleting a note moves it to **Trash**, where it stays for **30 days** before permanent removal. Restore a note from **Trash** in the sidebar; it returns to its original folder.

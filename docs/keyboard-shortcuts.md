# Keyboard Shortcuts

Tessera is built to be driven from the keyboard. On macOS use **Cmd**; on Windows and Linux use **Ctrl**. View this list anytime by pressing **?**.

## Essentials

- **Cmd/Ctrl + K** — open the command palette
- **Cmd/Ctrl + N** — new note
- **Cmd/Ctrl + Shift + F** — full search
- **Cmd/Ctrl + Shift + I** — open the Inbox
- **Cmd/Ctrl + ,** — open settings
- **?** — show this shortcuts list

## Navigation

- **Cmd/Ctrl + \\** — toggle the sidebar
- **Cmd/Ctrl + .** — toggle focus mode
- **Cmd/Ctrl + [** — go back
- **Cmd/Ctrl + ]** — go forward
- **Cmd/Ctrl + Shift + O** — jump to a heading in the current note (outline)

## Editing

- **Cmd/Ctrl + B** — bold
- **Cmd/Ctrl + I** — italic
- **Cmd/Ctrl + U** — underline
- **Cmd/Ctrl + Shift + S** — strikethrough
- **Cmd/Ctrl + L** — turn the selection into a link
- **Cmd/Ctrl + E** — inline code
- **Cmd/Ctrl + Shift + C** — copy the current code block

## Blocks

- **/** — open the slash menu to insert a block
- **Cmd/Ctrl + Shift + ↑ / ↓** — move the current block up or down
- **Cmd/Ctrl + D** — duplicate the current block
- **Cmd/Ctrl + Shift + Backspace** — delete the current block
- **Tab / Shift + Tab** — indent or outdent a list item

## Tasks

- **Cmd/Ctrl + Shift + T** — quick-add a task
- **Cmd/Ctrl + Enter** — toggle the current checkbox between done and not done

## Comments and history

- **Cmd/Ctrl + Shift + M** — add a comment to the selection
- **Cmd/Ctrl + Shift + H** — open version history

## Customizing shortcuts

Remap most shortcuts under **Settings → Preferences → Keyboard**. A handful of system shortcuts (the command palette and search) are fixed and cannot be changed. Reset to defaults at any time from the same screen.

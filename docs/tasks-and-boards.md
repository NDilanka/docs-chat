# Tasks and Boards

Tessera tracks to-dos as **tasks**. A task can live inline inside a note or be managed in a dedicated **board**, and the two stay in sync — checking it off in one place updates it everywhere.

## Creating tasks

- **Inline:** type `[] ` at the start of a line, or use the slash menu and choose **Task**.
- **Quick add:** press **Cmd/Ctrl + Shift + T** anywhere to capture a task without leaving your current note.
- **From a board:** click **+ Add task** in any board column.

## Task properties

Every task has these fields:

- **Title** (required)
- **Status** — `To do`, `In progress`, `Blocked`, or `Done`
- **Assignee** — any member of the space
- **Due date** — with an optional time and reminder
- **Priority** — `Low`, `Medium`, `High`, or `Urgent`
- **Tags** — one or more `#tags`

## Boards

A **board** is a Kanban view of the tasks in a space. Columns map to **Status** by default, but you can group by **Assignee**, **Priority**, or **Due date** instead. Drag a card between columns to change its value.

Create a board from **+ New → Board**. Each space can hold up to **20 boards**.

## Filtering and saved views

Filter a board by assignee, tag, priority, or due date. Combine filters with **AND** logic. Save a filtered board as a **view** (for example, "My urgent tasks this week") and switch between views from the tab bar at the top of the board.

## The Tasks page

The **Tasks** entry in the sidebar aggregates every task assigned to you across all spaces, grouped into **Today**, **Upcoming**, **No date**, and **Done**. This is your personal to-do list spanning the whole workspace.

## Recurring tasks

Set a task to repeat daily, weekly, monthly, or on a custom schedule. When you complete a recurring task, Tessera automatically creates the next occurrence with the same properties and a new due date.

## Reminders

Add a reminder to a due date and Tessera notifies you in the **Inbox**, by email, and via push notification on mobile. Reminders respect your account time zone, set under **Settings → Preferences**.

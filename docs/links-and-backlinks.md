# Links, Backlinks, and the Graph

Tessera turns a pile of notes into a connected knowledge base through links. Linking is how you build context instead of duplicating it.

## Creating a link

Type `[[` anywhere in a note to open the link picker, then start typing a note's name. Choose an existing note, or press **Enter** on a new name to create that note on the spot and link to it in one step.

You can also select text and press **Cmd/Ctrl + L** to turn the selection into a link.

## Backlinks

A **backlink** is the reverse of a link: if note A links to note B, then note B shows A in its **Backlinks** section. Backlinks appear in the right-hand context panel and at the bottom of the note. They update instantly and are never edited by hand, so you always see every note that references the one you're reading.

## Unlinked mentions

Below backlinks, Tessera lists **unlinked mentions** — notes that contain the current note's title as plain text but don't yet link to it. Click **Link** next to any mention to convert it into a real link.

## Link previews

Hover over a link to see a **preview card** with the target note's title and first lines. This lets you check a reference without navigating away.

## The graph view

Open **Graph** from the command palette to see your notes as a network of dots connected by links. Use it to spot clusters of related work and orphaned notes with no connections. Filter the graph by space or tag, and click any node to jump to that note. The graph view is available on the **Plus**, **Team**, and **Business** plans.

## Tags versus links

Use a **link** (`[[ ]]`) to connect to one specific note. Use a **tag** (`#topic`) to group many notes under a shared label. Open a tag to see every note and task that carries it. A note can have unlimited tags.

## Broken links

If a linked note is deleted, the link becomes a **broken link** shown in red. Tessera lists all broken links for a space under **Settings → Spaces → Maintenance** so you can fix or remove them.
